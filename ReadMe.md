# HSE-Python-Ind1  

Итоговое домашнее задание по предмету Программирование на Python

## Prerequisites

*   [Python](http://www.python.org) (>=3.10)
*   [Poetry](http://python-poetry.org) for dependency management

## Setup

Clone the repository and run the following commands to install the dependencies

```bash
make install
```

## Run

Run the server

```bash
make run
```
